Privacy Policy

Effective Date: March 25, 2026

1. Information Collection

Crystal Slice Content Manager does not collect, store, or share any personal data from TikTok users. The application only accesses the necessary TikTok API scopes (video.publish and user.info.basic) to perform its automation functions.

2. Data Usage

Any data accessed through the TikTok API is used solely for the purpose of publishing original video content to the authorized Crystal Slice account. We do not sell, trade, or transfer any user information to third parties.

3. Data Security

All automation processes are handled through a secure private server environment. We use industry-standard encryption to protect API tokens and credentials.

4. Third-Party Services

This application integrates with the TikTok API and n8n. Please refer to their respective privacy policies for more information on how they handle data within their platforms.

5. Contact Us

For any privacy-related inquiries regarding this private tool, please contact the Crystal Slice administrative team.
